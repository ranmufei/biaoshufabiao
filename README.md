- 管理界面 ： 列表 详细

- 采用分词算法 提取关键词； 提取一篇文章中的关键词

- 整理一个行业关键词库  如果文章是我要的目标行业 发送邮件通知我

- 升级 给予机器学习算法 读懂文章 如果是有用信息 发邮件通知
